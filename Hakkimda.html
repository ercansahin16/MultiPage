Hakkımda

Projelerim, yazılım geliştirme sürecindeki yeteneklerimi sergilemektedir. ASP.NET Core MVC, Entity Framework ve LINQ gibi teknolojileri kullanarak çeşitli web uygulamaları geliştirdim. Bunlar arasında API geliştirme, veri yönetimi ve kullanıcı arayüzleri oluşturma gibi projeler yer alıyor. Ayrıca, yazılımın mimari prensiplerini takip ederek sürdürülebilir ve ölçeklenebilir çözümler sunmaya özen gösteriyorum. GitHub profilimdeki bu projeler, profesyonel gelişimimin bir yansımasıdır.
